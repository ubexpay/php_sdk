## UbexPay PHP SDK
UbexPay cash management PHP SDK 

 Please visit API Docs on https://docs.ubexpay.com

## Installation
### Using Composer
Run **composer require ubexpay/php_sdk**  

### Manual  
Add **ubexpay/php_sdk** to the require section of composer.json
and run composer update.
